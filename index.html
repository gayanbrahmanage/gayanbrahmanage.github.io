Real-Time 3D-SLAM Package Development

2024: Developing a real-time 3D-SLAM package using C++ and GTK for Linux environments. The package processes images at 8 frames per second and includes a multi-threaded design with separate threads for depth prediction, local mapping, sparse bundle adjustment, and loop closing. It features a parallel loop processing structure with key-point detection, matching, and motion estimation using RANSAC and EM. The system dynamically allocates threads based on computational resources and integrates findings from my PhD research with existing literature.

Monocular Depth Prediction System

2024: Developing a monocular depth prediction system using Python and TensorFlow, which incorporates research findings from my PhD. The implementation features a Dual Convolution Vision Transformer (DCViT) for predicting depth from a single image using unsupervised learning methods. The primary objective of this project is to advance depth prediction accuracy and applicability in various fields such as autonomous driving and augmented reality.
